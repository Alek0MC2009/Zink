
# ⚡ Zink

**Zink** is a minimalist, high-performance dark-themed text editor designed for deep focus. Built with **Electron**, **React**, and **Tailwind CSS**, it was born from the need for a tool lighter than VS Code but more capable than the standard Windows Notepad.

> *Coded under the atmospheric influence of Beethoven's 'Moonlight Sonata'.*

---

## ✨ Features

- 🌑 **Zinc-Dark Interface:** An ultra-dark color palette based on `zinc-950` to reduce eye strain during long coding sessions.
- 📂 **History Management:** Quick access to your last 10 edited files directly from the sidebar.
- 🟠 **"Dirty" State Indicator:** A subtle orange dot notification system that lets you know if you have unsaved changes.
- 💾 **Smart File Creation:** If you start typing on an empty canvas, Zink automatically handles the "Save As" process for you on the first save.
- ⌨️ **Pro Shortcuts:** Native support for `Ctrl + S` and optimized system dialogs.
- 🔔 **Toast Notifications:** Elegant visual confirmations for every save action.

## 🚀 Tech Stack

Zink uses a modern stack to ensure speed and security:
- **Frontend:** [React](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
- **Runtime:** [Electron](https://www.electronjs.org/) (Multi-process IPC Architecture)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Feedback:** [React Hot Toast](https://react-hot-toast.com/)



## 🛠️ Installation & Setup

To run Zink locally or contribute to the project:

1. **Clone the repository:**
   ```bash
    git clone https://github.com/Alek0MC2009/Zink
    cd Zink
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run in development mode:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
